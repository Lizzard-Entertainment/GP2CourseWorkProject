CURRENTLY, GITHUB DOESN'T TRACK .LIB FILES, AND DOESN'T CARRY OVER ANY ENVIRONMENT PATH SETTINGS.

DO THIS BEFORE BUILDING PROJECTS AFTER DOWNLOADING A FRESH VERSION FROM GITHUB ONLINE REPO.

1) UNZIP LIBRARIES.ZIP INTO THE LIBRARIES FOLDER.  MERGE FOLDERS AND SKIP EXISTING FILES.

2.1) UNZIP THE DLL.ZIP FILE INTO THE DEBUG FOLDER IN THE PROJECT DIRECTORY.
2.1) ADD PATH=dll/ IN THE ENVIRONMENT SETTING IN VISUAL STUDIO.
